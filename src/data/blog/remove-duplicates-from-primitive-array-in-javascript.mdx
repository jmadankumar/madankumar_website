### In this Article, we will see how to remove duplicates from a number array using set

First we will declare and initialize number array which contains duplicate values


```javascript
const array = [20, 10, 20, 60, 3, 1, 10];
console.log(array);//[20, 10, 20, 60, 3, 1, 10]
```

now we will initialize a Set object using Set constructor and pass the array as a parameter

```javascript
const set = new Set(array);
console.log(set);//Set(5) {20, 10, 60, 3, 1}
```

you can see passing array to set remove all the duplicate number from the array and create a new Set Object

Now we will convert the Set object back to Array

```javascript
console.log(Array.from(set));//[20, 10, 60, 3, 1]
```

#### Final Code

```javascript
const array = [20, 10, 20, 60, 3, 1, 10];
console.log(array);//[20, 10, 20, 60, 3, 1, 10]
const set = new Set(array);
console.log(set);//Set(5) {20, 10, 60, 3, 1}
console.log(Array.from(set));//[20, 10, 60, 3, 1]
```
